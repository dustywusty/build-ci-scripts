# iMessage-notify

./iMessage-notify -s build-bot@email.com -r super@team.com,cool@team.com,member@team.com -m "[Build, FAILED] $JOB_NAME - $BUILD_URL"

-s originating service account
-r list of comma seperated recipients, no spaces
-m message
